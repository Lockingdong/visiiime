# visiiime-design-dashboard

## Priority 
```
1. git checkout develop

2. npm install

```

## Workflow
```
1. git checkout develop

2. 開發
    1. npm run serve
    2. 進到 http://localhost:8081/
    3. 開始開發

3. 開發完成
    1. ctrl + C 
    2. npm run build
    3. git add .
    4. git commit -m "commit 名稱"
    5. git push (可能要解衝突
    6. 到dev看成果
```

## Project setup
```
npm install
```
